# Heart Failure Prediction using Machine Learning

## Project Overview
This project predicts whether a patient is at risk of heart disease using Machine Learning algorithms.

## Dataset
- Heart Disease Dataset
- File: heart.csv

## Machine Learning Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)

## Best Model
Random Forest Classifier

Accuracy: 88.04%
